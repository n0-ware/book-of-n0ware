# rdesktop
###### Tags[^1]

The `rdesktop` command is a *Linux* utility to remote desktop connection (RDP) on another system in order to access the system desktop via a graphical user inferface. 

## Syntax
```
C:\>rdesktop -u [user] -p [password] [target_ip]
```
